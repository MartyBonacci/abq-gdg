## SpecSwarm

- **GitHub**: [github.com/MartyBonacci/specswarm](https://github.com/MartyBonacci/specswarm)
- **Installation**: `/plugin install specswarm@MartyBonacci`

Core commands:
- `/specswarm:build` - Build features from specs
- `/specswarm:fix` - Fix bugs with test-driven approach
- `/specswarm:ship` - Quality-gated merge workflow
- `/specswarm:upgrade` - Dependency and framework upgrades
- `/specswarm:refactor` - Code quality improvements

### The Four Minds Pattern

A multi-instance orchestration pattern with four specialized Claude Code roles:

1. **Developer** - Focused execution, writes the code
2. **Peer** - Reviews, collaborates, catches issues
3. **Mentor** - Broader context, guides the developer
4. **Highest Self** - Strategic planning, big picture decisions

- **Repository**: [github.com/MartyBonacci/four-minds-pattern](https://github.com/MartyBonacci/four-minds-pattern)
