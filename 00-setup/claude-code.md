# Claude Code Setup

## What It Is

Claude Code is Anthropic's official CLI for Claude. It's a terminal-based AI coding assistant that can read, write, and execute code in your projects.

## Installation

```bash
npm install -g @anthropic-ai/claude-code
```

Or with Homebrew:

```bash
brew install anthropic/claude/claude-code
```

## First Run

```bash
claude
```

You'll be prompted to authenticate with your Anthropic API key.

## Documentation

- [Official Docs](https://docs.anthropic.com/claude/claude-code)
- [GitHub Repository](https://github.com/anthropics/claude-code)

## Quick Tips

- Use `/help` to see available commands
- Claude Code maintains conversation context across your entire session
- It can use MCP (Model Context Protocol) servers for extended capabilities
